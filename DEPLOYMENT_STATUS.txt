📞 TROUBLESHOOTING

  Server Not Starting?
    → Check if port 8080 is in use: netstat -ano | findstr 8080
    → Try different port: php -S localhost:8081

  Database Connection Failed?
    → Verify MariaDB is running
    → Check application/config/database.php credentials
    → Run: mysql -u root bookingsoftware -e "SELECT 1;"

  Export Not Working?
    → Verify application/cache/ is writable
    → Check PHP memory_limit is sufficient
    → Look in application/logs/ for errors

  Permission Denied?
    → Check file ownership
    → Verify application/logs/ directory permissions
    → Run: icacls application\cache /grant Users:F
